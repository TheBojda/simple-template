Simple template

Simple template is a HTML templating system implemented in Java. Simple template can be used on the server side (ex: in a servlet on AppEngine), or the client side with GWT. The base conception of simple template is programming logic-free plain HTML templating.

How simple template is working?

Simple template templates are plain HTML documents, where sections are defined with HTML comments (<!--{section name}-->...<!--{/section}-->) or simply with the {name} form (it is equal to <!--{section name}--><!--{/section}-->). A template is something like this:

<html>
    <head>
        <title>Test</title>
    </head>
    <body>
        <ul>
            <!--{section container}-->
                <!--{section odd}-->
                <li><i>{item}</i></li>
                <!--{/section}-->
                <!--{section even}-->
                <li><b>{item}</b></li>
                <!--{/section}-->
            <!--{/section}-->
        </ul>
    </body>
</html>

This sections are reusable parts of the template, and empty slots for the data. The first step of using templates are parsing them:

Template t = Template.parse(... template string ...);

Every section is accessible by name, and can be used as a new template.

Template container = t.get("container");
Template odd = container.get("odd");
Template even = container.get("even");

These Template objects are reusable structures for generating content. The content generation is done by the template renderer, which is accessible by the getRenderer() method.

TemplateRenderer tr = t.getRender();

The TemplateRenderer objects .set() method is used to fill the template.

odd.getRender().set("item", "item" + i)

This code will generate something like this:

<li><i>item0</i></li>

It is possible to fill the slot with sequence of strings by the addTo() method:

tr.addTo("container","test1");
tr.addTo("container","test2");

This code will generate something like this:

...
<ul>
test1
test2
</ul>
...

If the renderer is filled, the content can be generated by the render() method. Look at a full example by using simple template:

Template t = Template.parse(... template source ...);
Template container = t.get("container");
Template odd = container.get("odd");
Template even = container.get("even");

TemplateRenderer tr = t.getRender();
for (int i = 0; i < 10; i++)
  tr.addTo("container",
    (i % 2 == 0 
    ? 
    odd.getRender().set("item", "item" + i)
    : 
    even.getRender().set("item", "item" + i)
    ));
System.out.println(tr.render());
